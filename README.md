## Installation Guide

이 프로젝트의 정보를 이용하려면, 다음 안내에 따라 진행하십시오.

### How to download project source code

소스코드 관리 시스템에서 다음 명령으로 프로젝트를 내려 받습니다.

```bash
$ git clone git@github.com:namjunemy/typescript_backend.git
$ cd typescript_backend
```
